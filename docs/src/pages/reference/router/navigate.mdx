# navigate

Programmatically navigate to routes with dynamic parameters, query strings, and history control.

## API

```typescript
function navigate(
  pattern: string,
  params?: Record<string, string>,
  query?: Record<string, string>,
  options?: { replace?: boolean; hash?: boolean }
): void
```

## TypeScript

The function is fully typed for parameters and options.

```typescript
// Navigate to /users/42?sort=asc
navigate(
  '/users/:id', 
  { id: '42' }, 
  { sort: 'asc' }
);
```

## Basic Usage

Use `navigate` to trigger navigation from event handlers, effects, or anywhere in your application logic.

```typescript
import { navigate } from "@hellajs/router";

// Navigate to a static path
navigate("/dashboard");

// Substitute route parameters
// Navigates to /posts/hello-world
navigate("/posts/:slug", { slug: "hello-world" });

// Add query parameters
// Navigates to /search?q=reactive
navigate("/search", {}, { q: "reactive" });
```

## Key Concepts

**History Management**: `navigate` integrates with the browser's History API, pushing new entries by default or replacing them with the `replace` option.

**Parameter Substitution**: Route patterns with `:param` placeholders are dynamically substituted with values from the params object.

**Query String Handling**: Query parameters are automatically serialized and appended to the final URL.

## Important Considerations

- Navigation is **asynchronous** - the route change isn't immediate
- Using `replace: true` prevents users from navigating back to the previous page
- Invalid parameter keys (not matching route pattern) are ignored
- Navigation triggers a full route evaluation and component re-render

## Use Cases

**Conditional Navigation**:
```typescript
// Navigate based on user state
if (user.isAuthenticated) {
  navigate("/dashboard");
} else {
  navigate("/login", {}, {}, { replace: true });
}
```

**Form Submission Redirects**:
```typescript
// After successful form submission
const handleSubmit = async (data) => {
  await saveUser(data);
  navigate("/users/:id", { id: data.id }, {}, { replace: true });
};
```

**Programmatic Route Changes**:
```typescript
// Navigate with query params for filtering
navigate("/products", {}, { category: "electronics", sort: "price" });
```
