## Quick Reference

### Essential APIs for Getting Started

- [`signal()`](/api/core/signal) - Create reactive state
- [`computed()`](/api/core/computed) - Derive values from state
- [`effect()`](/api/core/effect) - Handle side effects
- [`mount()`](/api/dom/mount) - Render components to the DOM

### Key Concepts to Understand

- [Reactivity](/concepts/reactivity) - How HellaJS tracks dependencies
- [Templating](/concepts/templates) - JSX patterns and best practices
- [Component Architecture](/concepts/components) - Building reusable components

## Prerequisites

Before starting these guides:

- **JavaScript knowledge**: ES6+, destructuring, arrow functions
- **Basic HTML/CSS**: Understanding of DOM and styling
- **Build tools**: Familiarity with npm/yarn and bundlers (helpful)
