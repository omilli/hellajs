Get HellaJS running in your project in minutes. Choose your preferred build tool and follow the setup guide.

## Quick Start

The fastest way to try HellaJS is with Vite:

```bash
npm create vite@latest my-hella-app -- --template vanilla
cd my-hella-app
npm install
npm install @hellajs/core @hellajs/dom
npm install --save-dev vite-plugin-hellajs
```

Configure Vite in `vite.config.js`:

```js
import { defineConfig } from 'vite';
import viteHellaJS from 'vite-plugin-hellajs';

export default defineConfig({
  plugins: [viteHellaJS()]
});
```

## Build Tool Setup

### Vite (Recommended)

Install the plugin and configure:

```bash
npm install --save-dev vite-plugin-hellajs
```

```js
// vite.config.js
import { defineConfig } from 'vite';
import viteHellaJS from 'vite-plugin-hellajs';

export default defineConfig({
  plugins: [viteHellaJS()]
});
```

### Rollup

Install the Rollup plugin:

```bash
npm install --save-dev rollup-plugin-hellajs
```

Configure in `rollup.config.js`:

```js
import rollupHellaJS from 'rollup-plugin-hellajs';

export default {
  // ... other config
  plugins: [rollupHellaJS()]
};
```

### Babel

Install the Babel preset:

```bash
npm install --save-dev @hellajs/babel-preset
```

Configure in `.babelrc` or `babel.config.js`:

```json
{
  "presets": ["@hellajs/babel-preset"]
}
```

## TypeScript Setup

For TypeScript projects, use the TypeScript template and add types:

```bash
npm create vite@latest my-hella-app -- --template vanilla-ts
cd my-hella-app
npm install
npm install @hellajs/core @hellajs/dom
npm install --save-dev vite-plugin-hellajs
```

Update your `tsconfig.json`:

```json
{
  "compilerOptions": {
    "jsx": "preserve",
    "types": ["@hellajs/dom"],
    "moduleResolution": "bundler",
    "target": "ES2022",
    "lib": ["ES2022", "DOM", "DOM.Iterable"]
  }
}
```

## Verification

Create a simple test to verify everything works:

```jsx
// main.js
import { signal } from '@hellajs/core';
import { mount } from '@hellajs/dom';

const App = () => {
  const count = signal(0);

  return (
    <button onclick={() => count(count() + 1)}>
      Clicked {count} times
    </button>
  );
};

mount(App, '#app');
```

Run your dev server:

```bash
npm run dev
```

Click the button - if the count updates automatically, you're all set!

## Troubleshooting

<details>
<summary>**Common Issues & Solutions**</summary>

**❌ `Uncaught ReferenceError: React is not defined`**

This happens when JSX transforms expect React. Make sure you're using the HellaJS plugins:
- ✅ Vite: `vite-plugin-hellajs`
- ✅ Rollup: `rollup-plugin-hellajs` 
- ✅ Babel: `@hellajs/babel-preset`

**❌ `Cannot read properties of undefined (reading 'mount')`**

Check your imports:
```jsx
// ✅ Correct
import { mount } from "@hellajs/dom";

// ❌ Common mistake
import { mount } from "@hellajs/core";
```

**❌ TypeScript errors with JSX**

Ensure your `tsconfig.json` has:
```json
{
  "compilerOptions": {
    "jsx": "preserve",
    "types": ["@hellajs/dom"]
  }
}
```

**❌ Components not updating when signals change**

Make sure you're passing the signal function, not calling it:
```jsx
// ✅ Correct - passes the signal function
<div>{count}</div>

// ❌ Wrong - calls the signal once, gets a static value
<div>{count()}</div>
```

</details>
