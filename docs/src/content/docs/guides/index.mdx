## Where to Start

**New to HellaJS?** Start with our [Complete Tutorial](/guides/getting-started/tutorial) to build your first application.

**Building a larger app?** Check out [Project Structure](/guides/getting-started/project-structure) and [State Patterns](/guides/patterns/state-patterns).

**Performance concerns?** Jump to our [Performance Guide](/guides/performance/optimization).

## Prerequisites

Before diving into these guides, make sure you're familiar with:

- **Core Concepts**: [Reactivity](/concepts/reactivity), [Templating](/concepts/templates)
- **Essential APIs**: [`signal()`](/api/core/signal), [`computed()`](/api/core/computed), [`mount()`](/api/dom/mount)

## Getting Help

If you get stuck while following any guide:

1. Check the [Troubleshooting](#troubleshooting) section in each guide
2. Review the related [API documentation](/api/)
3. Explore the [example applications](/examples)
4. Ask questions in our community discussions
