Get HellaJS up and running in your project. Choose the setup that matches your build toolchain.

## üöÄ Quick Start (Recommended)

The fastest way to start a new HellaJS project:

```bash
npm create vite@latest my-hella-app -- --template vanilla
cd my-hella-app
npm install
npm install @hellajs/core @hellajs/dom
npm install --save-dev vite-plugin-hellajs
```

Configure Vite in `vite.config.js`:

```js
import { defineConfig } from 'vite';
import viteHellaJS from 'vite-plugin-hellajs';

export default defineConfig({
  plugins: [viteHellaJS()]
});
```

Start building:

```jsx
// main.js
import { signal } from "@hellajs/core";
import { mount } from "@hellajs/dom";

const App = () => {
  const count = signal(0);
  return (
    <button onclick={() => count(count() + 1)}>
      Clicked {count} times
    </button>
  );
};

mount(App, "#app");
```

## üîß Build Tool Configuration

HellaJS JSX needs to be transformed for the browser. Choose your build tool:

### Vite (Recommended)

```bash
npm install --save-dev vite-plugin-hellajs
```

```js
// vite.config.js
import { defineConfig } from 'vite';
import viteHellaJS from 'vite-plugin-hellajs';

export default defineConfig({
  plugins: [viteHellaJS()],
  esbuild: {
    jsx: 'preserve' // Let the plugin handle JSX
  }
});
```

### Rollup

```bash
npm install --save-dev rollup-plugin-hellajs
```

```js
// rollup.config.js
import rollupHellaJS from 'rollup-plugin-hellajs';

export default {
  input: 'src/main.js',
  output: {
    file: 'dist/bundle.js',
    format: 'es'
  },
  plugins: [rollupHellaJS()]
};
```

### Babel

```bash
npm install --save-dev @hellajs/babel-preset
```

```json
// .babelrc or babel.config.js
{
  "presets": ["@hellajs/babel-preset"]
}
```

### Webpack

```bash
npm install --save-dev babel-loader @hellajs/babel-preset
```

```js
// webpack.config.js
module.exports = {
  module: {
    rules: [
      {
        test: /\.(js|jsx)$/,
        exclude: /node_modules/,
        use: {
          loader: 'babel-loader',
          options: {
            presets: ['@hellajs/babel-preset']
          }
        }
      }
    ]
  }
};
```

## üìù TypeScript Setup

For TypeScript projects, install types and configure properly:

### Install Dependencies

```bash
npm create vite@latest my-hella-app -- --template vanilla-ts
cd my-hella-app
npm install
npm install @hellajs/core @hellajs/dom
npm install --save-dev vite-plugin-hellajs
```

### TypeScript Configuration

```json
// tsconfig.json
{
  "compilerOptions": {
    "target": "ES2022",
    "lib": ["ES2022", "DOM", "DOM.Iterable"],
    "module": "ESNext",
    "moduleResolution": "bundler",
    "allowSyntheticDefaultImports": true,
    "esModuleInterop": true,
    "jsx": "preserve",
    "types": ["@hellajs/dom"],
    "strict": true,
    "skipLibCheck": true
  },
  "include": ["src/**/*"],
  "exclude": ["node_modules", "dist"]
}
```

### Type-Safe Components

```tsx
// App.tsx
import { signal, Signal } from '@hellajs/core';
import { mount } from '@hellajs/dom';

interface User {
  id: number;
  name: string;
  email: string;
}

const UserProfile = ({ user }: { user: Signal<User> }) => {
  const isEditing = signal(false);
  
  return (
    <div>
      <h2>{user().name}</h2>
      <p>{user().email}</p>
      <button onclick={() => isEditing(!isEditing())}>
        {isEditing() ? 'Save' : 'Edit'}
      </button>
    </div>
  );
};

const App = () => {
  const user = signal<User>({
    id: 1,
    name: 'John Doe',
    email: 'john@example.com'
  });

  return <UserProfile user={user} />;
};

mount(App, "#app");
```

## üåê CDN Usage

For quick prototyping or simple projects:

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="UTF-8">
  <title>HellaJS App</title>
</head>
<body>
  <div id="app"></div>
  
  <script type="module">
    import { signal } from 'https://esm.sh/@hellajs/core';
    import { mount } from 'https://esm.sh/@hellajs/dom';
    
    const App = () => {
      const count = signal(0);
      return h('button', {
        onclick: () => count(count() + 1)
      }, `Count: ${count}`);
    };
    
    mount(App, "#app");
  </script>
</body>
</html>
```


## üêõ Troubleshooting

### Common Issues & Solutions

**‚ùå `Uncaught ReferenceError: React is not defined`**

This happens when JSX transforms expect React. Ensure you're using HellaJS plugins:
- Vite: `vite-plugin-hellajs`
- Rollup: `rollup-plugin-hellajs` 
- Babel: `@hellajs/babel-preset`

**‚ùå `Cannot resolve '@hellajs/core'`**

Check your package.json dependencies:
```bash
npm list @hellajs/core
```

If missing, install it:
```bash
npm install @hellajs/core @hellajs/dom
```

**‚ùå TypeScript JSX errors**

Ensure your `tsconfig.json` includes:
```json
{
  "compilerOptions": {
    "jsx": "preserve",
    "types": ["@hellajs/dom"]
  }
}
```

**‚ùå Build errors with JSX**

Check that your build tool configuration loads the HellaJS plugin and that JSX files use `.jsx` or `.tsx` extensions.

<details>
<summary>üîç **More Troubleshooting Steps**</summary>

1. **Clear build cache**: `rm -rf node_modules/.cache dist`
2. **Reinstall dependencies**: `rm -rf node_modules package-lock.json && npm install`
3. **Check file extensions**: Use `.jsx` or `.tsx` for JSX files
4. **Verify plugin order**: HellaJS plugins should come first in build config
5. **Check browser console**: Look for error messages and stack traces

</details>
