# Routing

HellaJS Router provides reactive client-side routing using function-based route handlers that integrate with reactive signals.

## Basic Setup

```jsx
import { router, navigate } from '@hellajs/router';
import { signal } from '@hellajs/core';

const currentView = signal(null);

router({
  routes: {
    '/': () => currentView(<Home />),
    '/about': () => currentView(<About />),
    '/user/:id': (params) => currentView(<UserProfile userId={params.id} />)
  },
  notFound: () => currentView(<NotFound />)
});

const App = () => (
  <div>
    <nav>
      <button onclick={() => navigate('/')}>Home</button>
      <button onclick={() => navigate('/about')}>About</button>
    </nav>
    <main>{currentView}</main>
  </div>
);
```

## Route Parameters

Capture dynamic URL segments and query parameters:

```jsx
router({
  routes: {
    '/user/:id': (params) => {
      currentView(<UserProfile userId={params.id} />);
    },
    '/posts/:category/:slug': (params) => {
      currentView(<PostDetail category={params.category} slug={params.slug} />);
    },
    '/search': (params, query) => {
      currentView(<SearchResults term={query.q} category={query.category} />);
    }
  }
});
```

## Navigation

```jsx
import { navigate } from '@hellajs/router';

// Basic navigation
navigate('/dashboard');

// Replace history entry
navigate('/login', { replace: true });

// Go back
window.history.back();
```

## Route Guards

Protect routes with authentication:

```jsx
const requireAuth = () => {
  if (!localStorage.getItem('token')) {
    navigate('/login');
    return false;
  }
  return true;
};

router({
  routes: {
    '/dashboard': {
      before: requireAuth,
      handler: () => currentView(<Dashboard />)
    }
  }
});
```

## Configuration

```jsx
router({
  routes: {
    '/': () => currentView(<Home />),
    '/profile': '/user/me', // Simple redirect
  },
  hash: true, // Hash-based routing for static hosting
  notFound: () => currentView(<NotFound />)
});
```

## Integration Examples

### With Resources

```jsx
import { resource } from '@hellajs/resource';

const UserPage = ({ userId }) => {
  const user = resource(() => 
    fetch(`/api/users/${userId}`).then(r => r.json()),
    { key: () => userId }
  );
  
  return (
    <div>
      {user.loading() && <div>Loading...</div>}
      {user.data() && <h1>{user.data().name}</h1>}
    </div>
  );
};
```

### With State Management

```jsx
import { store } from '@hellajs/store';

const appStore = store({
  currentPage: null,
  setCurrentPage(page) { this.currentPage = page; }
});

router({
  routes: {
    '/dashboard': () => {
      appStore.setCurrentPage('dashboard');
      currentView(<Dashboard />);
    }
  }
});
```

## Best Practices

- Use meaningful, SEO-friendly paths
- Implement proper 404 handling
- Show loading states for data-heavy routes
- Keep route handlers focused and delegate complex logic to components
- Use route guards for authentication checks
