# route

A reactive signal containing current route information including path, parameters, query, and handler.

## API

```typescript
function route(): RouteInfo

interface RouteInfo {
  path: string;
  params: Record<string, string>;
  query: Record<string, string>;
  handler: RouteHandler | null;
}
```

## TypeScript

The `route` signal is strongly typed, providing autocomplete for its properties.

```typescript
import { route, type RouteInfo } from '@hellajs/router';

const currentRoute: RouteInfo = route();
```

## Basic Usage

Access the `route` signal inside a reactive context to get the current path, parameters, and query string.

```typescript
import { effect } from '@hellajs/core';
import { route } from '@hellajs/router';

effect(() => {
  const { path, params, query } = route();
  
  console.log(`Current path: ${path}`);
  
  if (params.id) {
    console.log(`User ID from params: ${params.id}`);
  }
  
  if (query.sort) {
    console.log(`Sort order from query: ${query.sort}`);
  }
});
```

## Key Concepts

**Reactive Signal**: `route()` is a reactive signal that automatically updates when the current route changes, triggering any dependent computations or effects.

**Route Information**: Provides comprehensive route data including the current path, extracted parameters from route patterns, and parsed query string values.

**Handler Reference**: Contains a reference to the active route handler function, useful for dynamic route behavior.

## Important Considerations

- **Performance Impact**: Accessing `route()` in frequently updating contexts can cause unnecessary re-renders
- **Route Parsing**: Parameters and query values are always strings - cast to other types as needed
- **Handler Timing**: The `handler` property may be `null` during route transitions
- **Reactive Context Required**: Must be used within reactive contexts (effects, computed) for automatic updates

## Use Cases

**Active Navigation Links**:
```jsx
const NavLink = ({ href, children }) => {
  const isActive = computed(() => route().path === href);
  return <a class={isActive() ? 'active' : ''}>{children}</a>;
};
```

**Conditional Content Rendering**:
```jsx
const UserProfile = () => {
  const { params } = route();
  const userId = params.id;
  
  return userId ? <Profile id={userId} /> : <NotFound />;
};
```

**Query-Based Filtering**:
```jsx
effect(() => {
  const { query } = route();
  const filter = query.category || 'all';
  updateProductList(filter);
});
```
